# LOF/ETF H5 公众号页面图标切图资产

包含 24 个从页面设计稿整理/重绘的独立图标资产：

- `svg/`：24×24 viewBox 矢量 SVG，透明背景
- `png/64/`：64×64 PNG，透明背景
- `png/128/`：128×128 PNG，透明背景
- `png/256/`：256×256 PNG，透明背景
- `manifest.json`：图标名称、用途、颜色说明

默认颜色遵循页面视觉规范：
- 主品牌色 `#127C74`
- 信息蓝 `#2F6FED`
- 风险橙 `#F08A24`
- 强调红 `#E24A4A`
- 稳定绿 `#2F9E69`

前端建议：SVG 可直接内联或通过 img 引用；需要主题换色时，可把 SVG 中的固定 `stroke` 替换为 `currentColor`。
