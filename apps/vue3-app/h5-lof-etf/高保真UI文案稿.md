# 基金套利 + 可转债机会助手 H5 高保真 UI 文案稿

更新日期：2026-04-25

## 1. 使用说明

本文档用于定义 H5 各页面的高保真展示文案，目标是统一：

- 页面标题
- 卡片标题
- 标签文案
- 按钮文案
- 状态文案
- 风险提示文案
- AI 解读文案

统一原则：

- 所有结论均为 `参考表达`
- 不出现明确买卖指令
- 不出现收益承诺
- 不出现诱导性下单语言

统一固定提示：

`以下内容基于历史数据、规则模型、公开信息与模型生成结果整理，仅供参考，不构成投资建议。`
